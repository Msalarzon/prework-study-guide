HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for HyperText Markup Language and is the standard language for creating web pages.
  • Web browsers use HTML to render web pages and display content to users.

CSS

  • CSS is used to style and format the visual appearance of HTML elements on a web page.
  • CSS can be applied to HTML elements using selectors and rules.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: fetches changes from a remote repository and merges them into the current branch
  • git push: pushes local changes to a remote repository
